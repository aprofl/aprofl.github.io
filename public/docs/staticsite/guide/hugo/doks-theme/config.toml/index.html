doks 의 설정 파일은 themes/doks/config/_default/Hugo.toml 입니다. Hugo는 프로젝트 루트의 config.toml 파일을 우선적으로 참조합니다. 만약 프로젝트 루트에 config.toml 파일을 새로 생성하면, Hugo는 이 파일을 주요 설정 파일로 사용합니다. config.toml 파일이 없는 경우에만 테마 디렉토리의 _default 설정 파일을 참조합니다. 따라서 프로젝트 루트에 config.toml 파일을 생성하면, _default에 있는 설정 파일을 참조하지 않고 새로 생성한 config.toml 파일을 사용하게 됩니다. 필요한 모든 설정을 이 파일에 포함시켜야 합니다. 하지만, 필요한 기본 설정을 프로젝트 루트의 config.toml 파일에 포함시키고, 추가 설정을 테마에서 가져오는 방식으로 사용할 수 있습니다. 이 경우, Hugo의 설정 병합 기능을 활용할 수 있습니다.
